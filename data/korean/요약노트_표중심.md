## 표 4. 개요 작성 - 「지침」 4대 규칙

| 번호 | 규칙 | 설명 | 함정 |
|------|------|------|------|
| 1 | 서론 = 배경 + 필요성/목적 | 작성 배경·사회적 추세·필요성 | 결론 항목이 서론에 들어가 있을 수 있음 |
| 2 | 본론은 2개 장으로 구성 | 보통 2장(원인/한계), 3장(해결 방안) | - |
| 3 | 본론 하위 항목 대응 | 2장 1절 ↔ 3장 1절, 2장 2절 ↔ 3장 2절 | 대응이 안 되는 항목이 들어가 있음 |
| 4 | 결론 = 기대 효과 + 향후 과제 | 1절 기대 효과, 2절 향후 과제 (이 순서 고정) | **본론에 들어갈 원인 분석이 결론에 들어가 있는 것이 가장 흔한 함정** |

## 표 5. 논리 추론 - 풀이 도구

| 도구 | 설명 | 예시 |
|------|------|------|
| 대우 (¬Q → ¬P) | 원 명제와 동치 | "A → B"의 대우 = "¬B → ¬A" |
| 선언지(P ∨ Q) + 부정 | 한쪽 부정 시 다른 쪽 확정 | "할인권 또는 체험권" + "할인권 X" → "체험권 ○" |
| 가정적 추론 | 가정 → 모순 → 부정 | "갑이 간다고 가정 → 모순 → 갑은 안 간다" |
| 삼단논법 | A→B, B→C 이면 A→C | "마라톤하는 사람 → 식단 조절 → 건강" 도출 |
| 역(逆)과 대우 구분 | 역은 동치 아님, 대우만 동치 | "A→B"의 역 "B→A"는 별개 명제 |

## 표 15. 음운 변동 4가지

| 유형 | 정의 | 예시 |
|------|------|------|

## 표 17. 한국어 사동/피동 형태

| 구분 | 형태 | 예시 |
|------|------|------|
| 사동 단형 | 접미사 -이-/-히-/-리-/-기-/-우-/-구-/-추- | 죽이다, 입히다, 울리다, 웃기다, 깨우다, 솟구다 |
| 사동 장형 | -게 하다 | 죽게 하다, 입게 하다 |
| 피동 단형 | 접미사 -이-/-히-/-리-/-기- | 보이다, 잡히다, 들리다, 안기다 |
| 피동 장형 | -아/어지다 | 만들어지다, 행해지다 |
