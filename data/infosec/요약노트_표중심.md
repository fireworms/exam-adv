## 표 2. 암호 알고리즘 분류 (대칭 vs 비대칭)

| 구분 | 알고리즘 |
|------|----------|
| **대칭키 블록** | DES, 3DES, AES, **SEED, ARIA**, IDEA, **LEA**, Blowfish, Twofish |
| **대칭키 스트림** | RC4, **A5/1**, A5/2, ChaCha20 |
| **비대칭 인수분해** | **RSA**, Rabin |
| **비대칭 이산대수** | **ElGamal, DSA, Schnorr, 디피-헬만** |
| **비대칭 타원곡선** | **ECC, ECDSA, ECDH** |
| **공개키 함정** | **배낭암호** (NP-완전 기반, 거의 취약) |
| **❌ 함정** | 디피-헬만 ≠ 대칭, ECC ≠ 대칭, A5/1 ≠ 공개키 |

## 표 3. 블록 암호 운영 모드 5가지 (최빈출)

| 모드 | 식 / 구조 | 패턴 노출 | 병렬(암호화) | 병렬(복호화) | 오류 전파 | IV/Nonce | 핵심 키워드 |
|------|-----------|-----------|------------|------------|-----------|----------|--------------|
| **ECB** | Cᵢ = E_K(Pᵢ) | **O (취약)** | O | O | 해당 블록만 | 불필요 | 같은 평문→같은 암문 |
| **CBC** | Cᵢ = E_K(Pᵢ⊕Cᵢ₋₁) | X | **X (직렬)** | O | Cᵢ오류→Pᵢ,Pᵢ₊₁ | 필수 | 이전 블록 영향 |
| **CFB** | Cᵢ = E_K(Cᵢ₋₁)⊕Pᵢ | X | X | O | 오류→해당+다음 | 필수 | 스트림화, 자기 동기 |
| **OFB** | Oᵢ = E_K(Oᵢ₋₁), Cᵢ=Oᵢ⊕Pᵢ | X | X | X | **오류 비전파** | 필수 | 오류 한 비트만 |
| **CTR** | Cᵢ = E_K(Nonce‖i)⊕Pᵢ | X | **O (완벽)** | **O** | 해당 블록만 | Nonce+카운터 | 병렬화 가능, 무작위 접근 |

**시험 패턴**:
- "동일 평문 동일 암호문 + 오류 전파" → OFB만 모두 해결
- "병렬 처리 + 카운터" → CTR
- "이전 블록과 XOR" → CBC

## 표 4. 접근 통제 모델 4가지

| 모델 | 정책 | 중심 | 규칙 | 적용 |
|------|------|------|------|------|
| **DAC** (임의적) | 소유자 결정 | 자율 | ACL, UNIX 권한 | 일반 OS |
| **MAC** (강제적) | 시스템 결정 | 보안 등급 | No Read Up/Down | 군사 |
| **BLP** | 강제적 | **기밀성** | **No Read Up, No Write Down** | 군사 |
| **Biba** | 강제적 | **무결성** | **No Read Down, No Write Up** | 시스템 |
| **Clark-Wilson** | - | **상업 무결성** | 트랜잭션, 직무분리, CDI 검증 | 금융 |
| **Chinese Wall** | - | **이해충돌** | 경쟁사 동시 접근 금지 | 컨설팅 |
| **RBAC** | - | 역할 | 역할에 권한 할당 | 기업 |

**기억법**: "기밀=BLP, 무결=Biba, 상업무결=Clark, 충돌=Chinese"

## 표 15. 위험 분석 접근법 + 위험 처리

**위험 분석**:
| 방법 | 특징 |
|------|------|
| **기준선** | 표준 체크리스트 |
| **비정형** | **전문가 지식·경험** |
| **상세** | 자산·위협·취약점 평가 |
| **복합** | 고위험=상세, 기타=기준선 |

**정성적 vs 정량적**:
- 정성적: **델파이법**, 시나리오, 순위결정, 브레인스토밍 (전문가 토론)
- 정량적: 과거자료, 확률분포, 수학공식, **ALE = SLE × ARO**

**위험 처리**:
| 방법 | 정의 |
|------|------|
| **수용** | 받아들이고 손실 감수 |
| **감소** | 통제·대책 적용 |
| **회피** | 행위·시스템 수행 안 함 |
| **전가** | **보험, 외주 계약** |
